# CodeAlpha_NetworkSniffer
A cross-platform network sniffer built using Python and Scapy that captures and analyzes network packets (IP, TCP/UDP, ports, and size).

## 📌 Description
This project is a basic network sniffer built using Python and Scapy.

## 🚀 Features
- Captures live network packets
- Displays Source and Destination IP
- Shows TCP/UDP protocols
- Displays port numbers
- Shows packet size

## 🛠️ Technologies Used
- Python
- Scapy
- Npcap

## ▶️ How to Run
1. Install dependencies:
   pip install scapy

2. Run as Administrator:
   python sniffer.py

## 📊 Example Output
Displays live packet information from your network.

## ⚠️ Important Note
Npcap must be installed for packet sniffing on Windows.
